The curriculum

Your learning path 🗺️

Twenty-two bite-sized lessons across four levels, starting before code even begins. Each one has plain-English explanations, runnable examples, and exercises with hidden solutions. Finish a lesson, hit “Mark lesson complete,” and watch the bar fill up.

🍵 A good rhythm One lesson per sitting. Type the examples out yourself (no copy-paste; fingers remember better than eyes), do the exercises, then take the level quiz when you finish a level. No installs yet? Every example runs in the Rust Playground.
