## The code exception, on purpose

Two carve-outs, both licensed under **MIT OR Apache-2.0** (see `LICENSE-MIT`
and `LICENSE-APACHE`) instead of CC BY-NC-SA:

1. **The software that runs the site:** the Rust web server (`src/`), the API
   functions (`functions/`), the site JavaScript and CSS (`docs/assets/`), and
   the tooling (`tools/`).
2. **Every code example, exercise solution, and puzzle program inside the
   course.** Students should be able to copy any snippet from any lesson
   straight into their own projects, commercial or not, without thinking about
   licensing. That is the point of teaching.
